# Hi, I'm Piero 👋

**Applied / Forward-Deployed AI Engineer | Lima, Peru (UTC-5)**

I build and deploy LLM agents and RAG systems into production, focusing on what makes AI actually usable in the real world: grounding answers in real data, measuring quality with eval-driven development, and handling failovers gracefully.

### 🛠️ Tech Stack
* **AI & Orchestration:** Anthropic Claude, LangGraph, RAG Architectures, Agentic Workflows.
* **Backend:** Python, FastAPI.
* **Data & Vectors:** PostgreSQL, pgvector, Supabase, Upstash.
* **DevOps:** Docker, GitHub Actions (CI/CD), Railway.

### 🚀 Featured Production Work
* [WhatsApp Clinic Agent](https://github.com/pierobruncoronado/whatsapp-clinic-agent): 24/7 RAG agent with zero-hallucination guardrails and no-PII logging.
* [Analyst SQL Agent](https://github.com/pierobruncoronado/analyst-sql-agent): Self-correcting LangGraph agent for business databases with strict read-only execution.
* [Semantic LLM Gateway](https://github.com/pierobruncoronado/semantic-llm-gateway): Semantic caching gateway for LLM APIs using Upstash Vector to optimize latency and costs.
